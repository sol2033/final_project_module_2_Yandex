Анализ лояльности пользователей Яндекс Афиши с помощью Python

Цель проекта: провести анализ поведения пользователей для команды маркетинга с целью выявления признаков, которые влияют на вероятность возврата пользователя на платформу.

Данные: с сервиса Яндекс Афиша (сервис для развлекательных мероприятий), выгруженные с реляционной БД с помощью SQL запросов и оформленные в виде датафреймов в pandas на Python. 
Этапы проекта:
1) Загрузка данных и их оценка
2) Предобработка данных 
3) Создание профиля пользователя и выявление выбросов
4) Исследовательский анализ данных
5) Выявление корреляции между количеством заказов пользователя и его признаками
6) Общие выводы по проекту и рекомендации заказчику
7) Публикация работы на GitHub

Итоги: был изучена полученная база данных, предобработана и исследована для решения исходной задачи. На основе исследования даны рекомендации заказчику и сделаны основные выводы, которые требовал заказчик
